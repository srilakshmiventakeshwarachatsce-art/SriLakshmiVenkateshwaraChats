TO REPLACE A PHOTO (e.g. with a real photo of your own food)
- Save the new photo with the SAME file name (all lowercase, hyphens).
- Keep it roughly square (e.g. 820x820) and under ~250 KB so the page stays fast.
  You can shrink at https://squoosh.app.
- The website updates automatically once the file is replaced.
